Detailed Description

Class representing an OpenGL shader program. Simplifies the initialization and management of all sources and bindings.

int Program::addShader ( std::string  fName,
int  type 
)

Associate a shader source file with the program as a Shader object. This will populate the Shader with its string-parsed source, but init() must still be called to compile and attach the shader to the program.

This function will return 0 upon error and automatically remove the failed shader from the program's list of Shaders.

Parameters
fName- string representation of the shader filename
type- GLEW-defined constant, one of: GL_VERTEX_SHADER, GL_FRAGMENT_SHADER, or GL_GEOMETRY_SHADER
Returns
1 on success or 0 on error

Completes the linking of the program with all attached shaders. Automatically validates the program and displays the info log if the info log is not empty.

Returns
